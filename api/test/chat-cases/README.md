# 小白熊行程助手聊天测试用例

此目录保存部署前必须通过的聊天回归用例。用例分为两层：

1. **聊天响应**：验证用户话语被正确识别为写操作、只读查询或普通聊天。
2. **工具执行**：模拟用户确认工具后，执行生产代码中的同一套 handler，并断言行程数据确实正确新增、修改或删除。

当前覆盖：

- 出行物品新增、删除。
- 预定项修改。
- 花销新增、修改、删除。
- 指定日期行程、未完成预定、途经地点和花销总额查询。
- 普通陪聊和景点推荐。
- 问候语误触发删除的回归保护。

可读用例及测试数据在 `cases.js`，自动执行器在 `chat-cases.test.js`。运行：

    npm test

测试仅操作内存中的行程副本，不访问 Azure，也不会修改线上数据。
